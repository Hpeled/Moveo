# Moveo project

Online web application to share pieces of code between Mentor and Student.

## Run Locally

Go to the api directory

```bash
  cd api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run devStart
```

Go to the client directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
